## 🗺️ Проект: Классификация типов лесного покрова (Forest Covertype)

**Датасет:** Forest Covertype из UCI (встроен в `sklearn.datasets.fetch_covtype`).  
**Задача:** по 54 характеристикам участка леса (высота, уклон, тип почвы, затенённость и т.д.) предсказать, какой из 7 видов деревьев на нём растёт.  
**Почему это сложно:**
- 581 012 записей (но для реальности возьмём подвыборку 50 000).
- 54 признака — нужно делать отбор.
- 7 классов — многоклассовая классификация.
- KNN здесь не идеален, но именно это и нужно показать: как бороться с проблемами (медленно, требует масштабирования, подбор K).
